POST/GET variables automatically made available to templates!

Once you send POST data via a form or GET data through the URL this will be automatically stored into a session and be made available to templates through the global template variable data. For example: 

                
                    {{ data.firstName }}

Have a look at the example form page. This will set the POST firstName and lastName variables. These will be available in the template through the data global variable. I.e. 

                
                    {{ data.firstName }}
                
            
                
                    {{ data.lastName }}

To create an array of elements just post more items with the same variable name and you will be able to retrieve it as a list in the template. I.e. 

                
                    {% for hobby in data.hobby %}
                      
{{ hobby }}

                    {% else %}
                      
You have no hobbies

                    {% endfor %}

Go to /reset-session to reset all the data saved in session.
